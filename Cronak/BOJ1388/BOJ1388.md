# 백준 1388번 - 바닥 장식 문제 풀이 설명 2026/01/31

## 문제 
입력 받은 타일이 연속되어 있는지 찾고 그 갯수를 알아내는 문제

'-' 모양은 가로로 인접해 있으면 하나의 타일로 취급합니다.

'|' 모양은 세로로 인접해 있으면 하나의 타일로 취급합니다.

## 풀이

배열을 순회하면서 입력 받은 값이 '-'면 1 '|'면 2로 설정하여 타일을 구분할 수 있게 하였고
연속된 값인지 확인하기 위하여 
```cpp
Tiles[i][j-1]
Tiles[i-1][j]
``` 
같은 연산을 사용하였습니다. 
연속된 값인지 판단하여야 하는 이유는 타일 갯수를 구하는 문제이므로 
2개 이상으로 연속된 경우는 한 개의 타일인 것으로 가정하기 때문입니다.

그렇기에 새로운 타일 종류일 경우에만 카운트(answer++)을 증가시킵니다.


## 기타
시간 복잡도 :  $$O(N \times M)$$

크래프톤 알고리즘 3주차에 출제되었으나 풀지 못했던 문제입니다.
