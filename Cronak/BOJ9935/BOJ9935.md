# 백준 9935번 - 문자열 폭발 문제 풀이 설명 2026/01/31

## 문제 
입력 받은 문자열 중에서 폭발 문자열과 같은 문자열을 없애는 문제 

## 풀이

문자열과 폭발 문자열을 입력 받은 후 

폭발 문자열과의 비교를 위해서 문자열이 폭발 문자열의 크기를 넘을 때까지 스택에 계속해서 문자열을 쌓습니다.

크기를 넘게 되면 폭발 문자열의 길이 단위로 검사를 합니다. 이때 폭발을 시켜야 한다는 것을 가정하고 검사를 진행합니다.
하나라도 틀리게 된다면 폭발을 시키지 않지만 

폭발 문자열과 일치한다면 폭발을 진행합니다.

이 순서대로 스택에 넣는 것이 모두 종료되고 



## 기타
시간 복잡도 :  $$O(N \times M)$$

크래프톤 알고리즘 2주차에 출제되었으나 풀지 못했던 문제입니다.
